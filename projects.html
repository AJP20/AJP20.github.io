NibbleAt

NibbleAt is a React Native built app available on both the App Store and Google Play. NibbleAt utilizes Yelps API for populating open restaurant within a 10 mile radius from the user. We store users likes and dislikes in a PostgreSQL via a GraphQL API, this will enable us in the future to recommend restaurants based off the users input
